Генератор таблицы животных

Введите количество строк таблицы. В каждой строке будет по 6 животных (3 собаки и 2 кота) в последовательности:

dog dog dog cat cat dog
dog dog cat cat dog dog
dog cat cat dog dog dog
cat cat dog dog dog cat
cat dog dog dog cat cat

Результат будет выведен в консоль браузера (F12).
